# ExamForge - NTRCA Exam Preparation Platform

A full-stack learning platform built with **Django** (Backend) and **React** (Frontend), fully containerized using **Docker**.

## 🏗 Project Structure
- `/backend`: Django REST API
- `/frontend`: React + Vite application
- `docker-compose.yml`: Orchestration file to run both services
